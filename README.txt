Problem Statement:
This program converts a number into a different base (between 2 and 16).

Describe the Solution:
I used a recursive function to keep dividing the number by the base and printing the remainder. I also used an array to handle values above 9 (like A-F for hex).

Pros:
- Works for bases 2 through 16
- Simple and easy to understand

Cons:
- Only works up to base 16
- Recursion can be confusing at first
